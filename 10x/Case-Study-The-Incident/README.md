დიალოგის რომელ მონაკვეთში გამოიკვეთა Stage 2 - მისკომუნიკაცია
დიალოგის რომელ მონაკვეთში დაიწყო კონფლიქტი Stage 3 - (თავდაცვის რომელი ფორმები გამოვლინდა?)
გაიგეთ რა სოციალური მოთხოვნილებები აქვთ კონფლიქტის მხარეებს (დაიხმარეთ 5 ბაზისური კითხვა)
როგორ გაუმკლავდებოდით კონფლიქტს ა) გიორგის ბ) ლაშას როლებში. დაწერეთ უშუალოდ რა ფრაზებს გამოიყენებდით კონფლიქტის მხარეების ადგილზე. (დაიხმარეთ ACR მოდელი)

სცენარი: „ეს შენი პასუხისმგებლობა იყო“
კონტექსტი:
სტარტაპის ოფისი.
production incident მოხდა ღამით. მომხმარებლები წერენ.
დილით საგანგებო შეხვედრაა.

პერსონაჟები:

გიორგი — engineering manager

ლაშა — junior developer

გიორგი:
— ლაშა, იცი, რატომ ვართ აქ?
ლაშა:
— production-ზე შეცდომა გამოვიდა.
გიორგი:
— და ვინ შეეხო ამ მოდულს ბოლოს?
ლაშა:
— მე… მაგრამ—
გიორგი (აწყვეტინებს):
— საკმარისია.
(ოთახში სიჩუმეა.)

გიორგი:
— ეს მოდული კრიტიკულია. რატომ არ იყო ეს განხილული?
ლაშა:
— მცირე ცვლილება იყო. ტესტები გავუშვი.
გიორგი:
— „მცირე“ არ არსებობს production-ში.
ლაშა:
— მაგრამ code review-ში არაფერი თქვით.
გიორგი:
— იმიტომ, რომ არ ველოდებოდი ასეთ შეცდომას.

(ტონი მკაცრდება.)
ლაშა:
— მაშინ რატომ დამიტოვეთ ეს task-ი?
გიორგი:
— იმიტომ, რომ უნდა ისწავლო პასუხისმგებლობა.
ლაშა:
— ეს სწავლაა?
გიორგი:
— ეს შედეგია.

ლაშა:
— მთელი ღამე ვიმუშავე fix-ზე.
გიორგი:
— იმიტომ, რომ დღისით შეცდი.
ლაშა:
— მარტო მე არა, სისტემაც—
გიორგი:
— ნუ გადამიბრალებ გუნდს.

(ლაშა ხმას უწევს.)
ლაშა:
— მაშინ რატომ მეძახით გუნდს, თუ შეცდომაზე მარტო ვრჩები?
გიორგი:
— იმიტომ, რომ ახლა junior ხარ.
ლაშა:
— და როდის არ ვიქნები?

(პაუზა. გიორგი უყურებს.)
გიორგი:
— როცა მსგავს რაღაცებს აღარ გააკეთებ.
ლაშა:
— და თუ გავაკეთებ?
გიორგი:
— მაშინ აქ ვერ გაჩერდები.

ოთახში დაძაბულობაა.
ლაშა სკამს სწევს უკან.
ლაშა:
— გასაგებია.
გიორგი:
— არა, არ არის გასაგები. იმიტომ ვართ აქ.
ლაშა:
— ჩემთვის უკვე გასაგებია.
